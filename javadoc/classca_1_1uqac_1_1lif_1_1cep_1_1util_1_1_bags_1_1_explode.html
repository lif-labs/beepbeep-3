Detailed Description

A 1:m function provided by the Bags utility class.

Given a collection of size m, it returns as its m outputs the elements of the collection. It can be seen as the oppsite of ToArray, ToList and ToSet. The ordering of the arguments is ensured when the input collection is itself ordered.
